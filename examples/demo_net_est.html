Generate spike train data¶

Given the connectivity matrix and the background intensity rate as inputs, I generate spike train data from p processes under M experiment. The function simu_net is saved in simu_net.py

In [4]:
from simu_net import simu_net
T = [1000,1000,1000] # [200,500,300] # 
rho, rate, lag = 2, 1, 1
M = Beta.shape[0]
p = Beta.shape[1]

np.random.seed(123)
DATA = simu_net( T, # length of each experiment, array of length M
                M, # no. experiments
                p, # no. processes
                Beta, U, # connectivity matrix and the background intensity rate
                rho , rate, lag # parameter used for transition kernel 
               )
Spike = DATA['Spike'] # output Spike, the spike train data, is a dictionary of length M with its value a p x p matrix
Phi = DATA['Phi'] # output Phi, the transition matrix, is a dictionary of length M with its value a p x p matrix

Estimate Multi-Experiment Networks¶

The key function, net_est, for multi-experiment network estimation are saved in net_est.py. The function repeatedly calls the general lasso solver (see function spg_genlasso). Besides the key input of the spike train data (dictionary format), net_est requires input of similarity matrix, W, and the transition matrix, X. net_est outputs the estimated connectivity matrix and background intensity.

If W is None, the default is to get empirical cross-correlation based similarity matrix (by calling function getW). If X is None, the default is to calcualte the transition matrix using function getX.

Joint estimation using informative similarity weights (default)¶

In this part, we call net_est to jointly estimate networks under all experiments. By giving W calculated from the cross-correlations, and setting up property tuning parameter values for the fusion penalty, lam2, the function estimate the networks jointly.

# plot estimated connectivity matrix

fig, axs = plt.subplots(4, 3,figsize=(60,45))
plt.axis('off')
for m in range(M):
    
    BETA_PLOT = np.zeros( (p,p))
    
    idx =  BETA_EST[m] * Beta[m] > 0 # consistent edge
    BETA_PLOT[idx] = 1
    axs[0,m].matshow( BETA_PLOT )
    axs[0,m].set_title('consistent est.', fontsize=15)
    #print(np.sum(idx) )
    
    idx =  BETA_EST[m] * Beta[m]  < 0 # differ in sign 
    BETA_PLOT[ idx ] = - 1
    axs[1,m].matshow( BETA_PLOT )
    axs[1,m].set_title('+diff in sign', fontsize=15)
    #print(np.sum(idx) )
    
    idx =   (BETA_EST[m]!=0) *   (Beta[m]==0)  # false discovery 
    BETA_PLOT[ idx  ] = - 1
    axs[2,m].matshow( BETA_PLOT )
    axs[2,m].set_title('+false positive', fontsize=15)
    #print(np.sum(idx!=0) )
    
    idx =   (BETA_EST[m]==0) *   (Beta[m]!=0)  # mis discovery 
    BETA_PLOT[ idx  ] = - 1
    axs[3,m].matshow( BETA_PLOT )
    axs[3,m].set_title('+false negative', fontsize=15)
    #print(np.sum(idx!=0) )
